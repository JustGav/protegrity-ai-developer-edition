## Contributing

We welcome community contributions! To add your solution:

1. Create a new directory under `community-solutions/` with a descriptive name.
2. Include a `README.md` with setup instructions and a description of your solution.
3. Submit a pull request following the guidelines in [CONTRIBUTING.md](../CONTRIBUTING.md).
