# Solutions

End-to-end solutions by Protegrity that combine multiple features (discovery + protection, discovery + redaction).

## Available Solutions

### find-and-protect

Combined discovery and protection flow — discover PII entities and protect them using tokenization-like operations.

- `sample-app-find-and-protect.py` — Python: discover + protect
- `sample-app-find-and-unprotect.py` — Python: unprotect previously protected data
- `sample-app-find-and-protect.sh` / `.bat` — Java wrappers
- `sample-app-find-and-unprotect.sh` / `.bat` — Java wrappers

### find-and-redact

Redaction pipeline — discover PII entities and redact or mask them.

- `sample-app-find-and-redact.py` — Python: discover + redact/mask
- `sample-app-find.py` — Python: discover and list PII entities
- `sample-app-find-and-redact.sh` / `.bat` — Java wrappers
- `sample-app-find.sh` / `.bat` — Java wrappers
